# ChurnShield

**Vehicle Insurance Churn Prediction with Explainable ML & Retention Strategy Dashboard**

ChurnShield is a data-driven web application that helps vehicle insurance companies identify at-risk customers, understand churn reasons, and design personalized retention strategies. It integrates **Machine Learning (XGBoost, SHAP)** with a **Flask-based dashboard** to deliver actionable insights for insurers.

---

## 🚀 Features
- **Customer Data Upload** → Single entry or bulk CSV upload.
- **Churn Prediction** → Predict churn probability using ML.
- **Explainable AI (SHAP)** → Interpret why a customer might churn.
- **Retention Strategy Generator** → AI + rule-based suggestions.
- **Visualization Dashboard** → Trends, KPIs, churn distribution, CLV.
- **Group-wise & Time-based Analysis** → Segment churn by policy type, region, etc.
- **PDF Report Generation** → Download insights and visuals for stakeholders.
- **Secure Login System** → Authentication and session handling.

---

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript, Jinja2
- **Machine Learning:** Scikit-learn, XGBoost, SHAP
- **Visualization:** Matplotlib, Seaborn
- **Database:** MySQL
- **Other Tools:** Pandas, NumPy, pdfkit, wkhtmltopdf, Google Gemini API (for AI-based strategies)
